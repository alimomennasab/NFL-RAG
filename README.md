# NFL RAG Chatbot
Q&A Gradio chatbot for NFL info (stats, rosters, coaching staffs, front offices) using a RAG pipeline: 
- MiniLM embeddings
- Pinecone vector storage and search
- Google Gemini API generated answers (context-only).
